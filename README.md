# Исследование результатов А/В-теста и поиск инсайтов

**Задачи проекта:** Используя данные A/B-теста мобильного инвестиционного приложения, проанализировать влияние обучающего онбординга на финансовое поведение пользователей и ключевые метрики монетизации.

**Описание проекта:** Проведён анализ результатов A/B-эксперимента по сравнению стандартного и обновлённого (с обучением рискам) онбординга. Оценено влияние на конверсию в первый и второй депозит, средние суммы инвестиций. Выявлено, что новый онбординг значимо повышает лояльность и конверсию во второй депозит (особенно среди пользователей высокорисковых активов), не снижая при этом конверсию в первый депозит. Для проверки статистической значимости различий использовался t-критерий Стьюдента и бутстрап-анализ. На основе результатов подготовлены рекомендации по внедрению обновлённого онбординга и его дальнейшей оптимизации.

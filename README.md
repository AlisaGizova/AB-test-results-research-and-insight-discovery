# **Анализ A/B-теста обучающего онбординга**

### **Задачи проекта**

- анализ влияния обновлённого онбординга на финансовое поведение пользователей
- оценка изменения ключевых метрик монетизации
- сравнение эффективности стандартного и обучающего подходов

### **Описание проекта**

Проведён анализ A/B-эксперимента в мобильном инвестиционном приложении. Исследовалось влияние двух типов онбординга:

- **Группа A:** стандартный онбординг
- **Группа B:** обновлённый онбординг с обучением рискам

### **Методология**

Для проверки статистической значимости результатов использовались:

- T-критерий Стьюдента
- Бутстрап-анализ

### **Ключевые результаты**

**Новый онбординг показал следующие эффекты:**

- значимо повысил лояльность пользователей
- увеличил конверсию во второй депозит
- особенно эффективен для пользователей высокорисковых активов
- не оказал негативного влияния на конверсию в первый депозит

### **Рекомендации**

1. внедрить обновлённый онбординг для всех пользователей
2. продолжить оптимизацию обучающих материалов
3. рассмотреть персонализацию контента для разных сегментов пользователей
